# AI-Based Fire Fighting and Human Detection Rescue Robot

## Project Overview
This project is an AI-based rescue robot capable of:
- Fire Detection
- Automatic Water Pump Activation
- Obstacle Detection
- Bluetooth Control
- Human Detection using Raspberry Pi
- Emergency Alert System

## Features
### Fire Detection
Detects fire using flame sensor and automatically activates the water pump.

### Obstacle Detection
Detects nearby obstacles using ultrasonic sensor and activates warning system.

### Human Detection
Uses Raspberry Pi camera and AI-based image processing to identify human presence.

### Bluetooth Control
Robot movement can be controlled using:
- F = Forward
- B = Backward
- L = Left
- R = Right
- S = Stop
